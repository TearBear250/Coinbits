# 🪙 CoinBits – Crypto for Kids!

**CoinBits** is an interactive educational web application that teaches children (ages 6–14) about Bitcoin and cryptocurrency in a fun, safe, and engaging way. No real money is involved — everything is a simulation for learning purposes.

## Features

| Module | Description |
|--------|-------------|
| 📚 **Learn** | Tabbed lessons on What is Crypto, Bitcoin Basics, Blockchain, Mining, Wallets, and Safety |
| ⛏️ **Mine** | Click-to-mine simulator — earn CoinBits by completing blocks |
| 👛 **Wallet** | View virtual balance, coin holdings, and transaction history |
| 📤 **Send & Receive** | Simulate sending CoinBits to friends with full validation |
| 🎯 **Quiz** | 8 multiple-choice questions with star ratings and answer review |

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) 18 or newer

### Install dependencies

```bash
npm install
```

### Run in development mode

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## Testing

```bash
npm test
```

Runs 24 tests across the Home, Mine, Quiz, and Transfer components using [Vitest](https://vitest.dev/) and [Testing Library](https://testing-library.com/).

## Tech Stack

- **Framework:** React 19 + Vite
- **Routing:** React Router DOM
- **Testing:** Vitest + Testing Library
- **Styling:** Pure CSS with CSS custom properties

## Safety

CoinBits uses **no real money** and requires **no accounts or personal information**. It is designed to be a parent-friendly educational tool that introduces children to cryptocurrency concepts safely.
